    # Interpreted Languages

    This module is all about taking those programming concepts and applying them to a specific language.

    Javascript is the interpreted language we'll be spending time with in this course. Mozilla's Javascript Basics offers a nice and concise reference of the programming concepts covered this far and how to invoke them in Javascript.

    Watch the following videos from freeCodeCamp's introduction to Javascript. Follow along with Beau by completing these steps. A video version is available on youtube:

    1. creating a new hello-webpage project on Glitch
    2. clicking the Show Live sunglasses in the top left
    3. opening the browser console for the Show Live tab
    4. code in the Glitch editor
